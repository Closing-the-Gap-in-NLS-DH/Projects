Contribute

This form is still under development, so bugs are likely to occur. If anything doesn’t work as expected, please open an issue on GitHub. Please make sure to provide as much information as possible for better reproducibility. We will continue to iterate on this form to include additional features, such as validation on entry, automatically acquired coordinates, and an option to load and edit existing JSON files.

To be implemented…

Most of the schema can be considered optional, but the following data points are required:

  • Name of the person adding the record
  • Project/organization name
  • Type (i.e., project or organization)
  • At least one website URL
  • Project/organization description
  • At least one associated place (usually a city)
  • Language(s) of the project/organization’s work (at least one)
  • Details for at least one contact person
  • Language(s) of sources with which the project/organization works (at least one)
  • Relevant keywords (chosen from a list)
